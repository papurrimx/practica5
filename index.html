HTML

HTML es el lenguaje de marcado que se utiliza para estructurar el contenido de las páginas web mediante etiquetas.

CSS

CSS permite definir la presentación visual de un sitio web, controlando colores, tamaños, fuentes y distribución.

JavaScript

JavaScript es un lenguaje de programación que añade interactividad y dinamismo a las páginas web.
